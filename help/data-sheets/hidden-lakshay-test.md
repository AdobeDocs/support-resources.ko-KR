Adobe Journey Optimizer의 이메일 Designer에서 컨텐츠 구성 요소를 사용하여 개인화되고 구조화된 이메일 레이아웃을 제작하는 방법에 대해 알아봅니다.

:::
![아이콘](https://cdn.experienceleague.adobe.com/icons/puzzle-piece.svg)

이메일 디자인을 위한 시각적 조각 활용

효율적인 이메일 디자인을 위해 시각적 조각을 사용하여 재사용 및 사용자 지정 가능한 구성 요소를 만드는 방법을 알아봅니다.

[시각적 조각에 대해 알아보기](../using/email/use-visual-fragments.md)

수신자 작업을 효과적으로 모니터링하기 위해 링크를 추가하고, 추적을 활성화하고, 추적 설정을 관리하는 방법을 알아봅니다.

[링크를 추가하고 메시지를 추적하는 방법에 대해 알아보기](../using/email/message-tracking.md)
:::

:::
![아이콘](https://cdn.experienceleague.adobe.com/icons/bullseye.svg)

Adobe Journey Optimizer의 의사 결정 관리를 사용하여 이메일 콘텐츠에서 개인화된 오퍼를 추가하고 미리 보는 방법에 대해 알아봅니다.

[개인화된 오퍼를 추가하는 방법 알아보기](../using/email/add-offers-email.md)

HTML 콘텐츠를 표시할 수 없을 때 호환성을 보장하면서 이메일의 일반 텍스트 버전을 만들고 편집하는 방법을 이해합니다.

[텍스트 버전을 생성하는 방법을 알아봅니다](../using/email/text-version-email.md)

![아이콘](https://cdn.experienceleague.adobe.com/icons/gear.svg)

사전 머리글, 문서 제목 및 언어 설정과 같은 메타데이터를 구성하여 이메일 가독성과 접근성을 향상시킵니다.

[메타데이터 추가 방법 알아보기](../using/email/email-metadata.md)
