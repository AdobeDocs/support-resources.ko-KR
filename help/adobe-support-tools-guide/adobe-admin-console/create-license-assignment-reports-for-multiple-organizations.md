# 여러 조직 및 제품에 대한 라이선스 할당 보고서 만들기

전역 관리자가 특정 날짜 범위에 대해 여러 조직 및 제품에 대한 세부 라이선스 보고서를 생성 및 다운로드하여 라이선스 프로비저닝의 정확한 추적을 용이하게 하는 방법에 대해 알아봅니다.

>[!NOTE]
>
>라이선스 할당 보고서를 만들고 보고 내보내려면 [Global Admin Console](https://global-admin-console.adobe.com/)에 로그인한 다음 **[!UICONTROL Insights]** > **[!UICONTROL 보고서]** > **[!UICONTROL 라이선스 할당]**&#x200B;으로 이동하십시오.

## 보고서 만들기

라이선스 할당 보고서를 통해 사전에 라이선스 프로비저닝을 모니터링하고 수동 추적을 줄일 수 있습니다. 글로벌 관리자는 여러 하위 조직의 제품 선택에 대해 라이선스 할당 보고서를 만들어 모든 부서에 걸친 소프트웨어 라이선스 프로비저닝 데이터를 모니터링할 수 있습니다.

1. Global Admin Console의 **[[!UICONTROL Insights]](https://global-admin-console.adobe.com/insights)** 탭으로 이동합니다.
1. **[!UICONTROL 라이선스 할당]** 페이지에서 **[!UICONTROL 보고서 만들기]**&#x200B;를 선택합니다.
1. 조직을 선택하고 **[!UICONTROL 다음]**&#x200B;을(를) 선택합니다. **[!UICONTROL 모두 선택]** 단추를 사용하여 각 조직을 개별적으로 선택하거나 상위 조직 내에서 모든 하위 조직을 선택할 수 있습니다.

   >[!NOTE]
   >
   >**특정 조직을 선택할 수 없는 이유를 알아보세요**:
   >하위 조직에 계약이 없거나 상위 조직과 동일한 제품을 가진 별도의 엔터프라이즈 계약이 있는 경우, 이 조직은 라이선스 할당 보고서를 생성할 수 없습니다. 예를 들어 상위 조직의 계약에 Adobe Acrobat이 있고 하위 조직의 계약이 다른 계약의 일부와 동일한 경우 제품은 할당으로 제한됩니다. 이로 인해 Global Admin Console에서의 보고서 생성도 제한됩니다. [각각의 Admin Console을 사용하여 이러한 조직의 프로비저닝을 추적하는 방법에 대해 알아봅니다](https://helpx.adobe.com/kr/enterprise/using/assignment-reports.html).

   >[!NOTE]
   >
   >활성 계약이 있는 조직에 대해서만 지정 보고서를 생성할 수 있습니다.

1. 보고서에 포함할 제품을 선택하고 **[!UICONTROL 다음]**&#x200B;을(를) 선택하십시오.

   >[!NOTE]
   >
   >**특정 제품을 선택할 수 없는 이유를 알아보세요**:
   >Global Admin Console에서 할당할 수 없는 제품은 보고서 작성에 포함되지 않습니다. 여기에는 현재 Workfront, Adobe Experience Manager, Adobe Experience Platform과 같은 일부 Digital Experience 제품과 Adobe Firefly Services, Acrobat Sign, Adobe Stock과 같은 제품도 포함되어 있습니다. [Adobe Admin Console을 사용하여 이러한 제품에 대한 라이선스 프로비저닝 데이터를 찾습니다](https://helpx.adobe.com/kr/enterprise/using/assignment-reports.html).

1. 보고서를 월별 또는 연도별로 집계할지 여부를 선택합니다.
1. 사용자 지정 날짜 범위를 선택하거나 사전 설정 옵션 중에서 선택합니다. 2020년 6월 18일부터 2020년 6월 18일까지, 계약 시작 날짜 이전에 시작하지 않는 한 그 전날까지의 시작 날짜를 선택할 수 있습니다.
1. 보고서를 CSV 파일로 내보내려면 **[!UICONTROL 다운로드]**&#x200B;를 선택하십시오.

보고서가 처리를 시작하고 **[!UICONTROL 라이선스 할당]** 페이지에 이름, 작성자, 작성 시간, 날짜 범위, 상태 등의 세부 정보와 함께 표시됩니다. 준비가 되면 이메일 알림을 받고 보고서가 자동으로 다운로드됩니다.

모든 전역 관리자는 보고서가 준비되면 언제든지 보고서 이름 옆에 있는 **[!UICONTROL 다운로드]** 아이콘을 사용하여 보고서를 다운로드할 수 있습니다.

>[!NOTE]
>
>- 최신 데이터가 반영되도록 하려면 할당 후 48시간 후에 보고서를 생성합니다.
>- 보고서는 생성된 후 90일 동안 유지됩니다.

## 이전에 생성된 보고서 보기 및 다운로드

전역 관리자는 조직의 다른 전역 관리자가 생성한 라이선스 할당 보고서를 보고 다운로드할 수 있습니다. 하지만 전역 뷰어에는 라이선스 할당 보고서 목록만 표시됩니다.

1. Global Admin Console의 **[[!UICONTROL Insights]](https://global-admin-console.adobe.com/insights)** 탭으로 이동합니다.
1. **[!UICONTROL 라이선스 할당]** 페이지에 모든 보고서가 다음 세부 정보와 함께 나열됩니다.

   | 필드 | 설명 |
   | ------------- | ------------------------------------------------------------------------------------------------- |
   | 이름 | 자동으로 생성되며 편집할 수 없습니다. |
   | 작성자 | 보고서를 생성한 전역 관리자입니다. |
   | 생성 시간 | 보고서를 만든 시스템 시간입니다. |
   | 날짜 범위 | 보고서에 대해 선택한 날짜 범위입니다. |
   | 상태 | 보고서를 다운로드할 준비가 되면 **성공**&#x200B;을, 아직 생성 중이면 **처리**&#x200B;을(를) 선택하십시오. |

1. 보고서를 CSV 파일로 내보내려면 보고서 옆에 있는 **[!UICONTROL 다운로드]** 아이콘을 선택합니다.

## 라이선스 할당 보고서 확인

다운로드하는 보고서에는 각 이벤트에 대한 다음 정보가 포함되어 있습니다.

| 필드 | 설명 |
| -------------------------------- | -------------------------------------------------------- |
| 조직 ID | 상위 조직에 연결된 고유 ID |
| 조직 이름 | 상위 조직 이름 |
| 하위 조직 ID | 선택한 하위 조직의 고유 식별자 |
| 하위 조직 이름 | 하위 조직 이름 |
| 라이선스 ID | 제품 라이선스에 대한 고유 ID |
| 제품 이름 | 선택한 제품의 이름 |
| 일자 | 날짜 선택 |
| 총 라이센스 수량 | 상위 조직 수준의 총 라이선스 수 |
| 하위 라이선스 | 하위 조직에 할당된 라이선스 수 |
| 월별/연간 최대 할당 수량 | 라이선스 프로비저닝 값 집계(월별/연간) |
